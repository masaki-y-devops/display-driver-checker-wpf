## 何をするアプリなのか

Windows向けのドライバ（特にディスプレイドライバを想定）の署名をチェックし、

真正性を確認できた場合のみ各ファイルのチェックサムを計算、

その後に計算結果をlinuxの``sha256sum -c``コマンドで検証可能な形で、

``(ファイル名).sha256``という形で保存します。

## 問題意識

こういう作業、めんどくさくないですか？

1. メーカーサイトから落としてきたドライバファイルの.exeファイルが破損していないかチェックする。

2. 今後リンク切れでダウンロードできなくなってしまうことを見越して、破損していないか確認してから光学ディスクなどにアーカイブする。

3. せっかくなのでメーカーがリリースしたままの真正な状態かを確かめてから後で破損の有無をチェックできるようにしておきたい。

というのも、GPD WIN MiniのAMD Radeon 780Mドライバが不安定で、バージョンを遡りながら逐一検証している際に思い立ちました。

各exeファイルをダウンロードさえすれば、その複数ファイルを本アプリのウインドウにD&Dしてぶち込むだけで署名真正性検証とチェックサム計算・保存がワンストップで実現しました。

## 使い方

.NET 10ランタイムが必要です。

ダウンロードはこちら→https://dotnet.microsoft.com/en-us/download/dotnet/thank-you/runtime-desktop-10.0.9-windows-x64-installer

起動して、任意のディスプレイドライバをウインドウ内に放り込んでください。

署名検証後、その.exeファイルと同階層に.sha256ファイルが作成されるはずです。
